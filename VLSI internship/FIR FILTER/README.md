# FIR Filter Design and Simulation

This task implements a digital FIR (Finite Impulse Response) filter using MATLAB.

## Description
- Input signal is filtered using predefined FIR coefficients.
- Convolution operation is used to generate the output.
- Output waveform is plotted to verify filtering behavior.

## Files
- fir_filter.m : MATLAB code for FIR filter
- Screenshots  : Simulation output and waveform

## Result
The FIR filter smooths the input signal and demonstrates stable, linear phase behavior.
